Retail Tracker
=============

Терминология:

- CRUD (Create Retrieve Update Delete)
- POS (Point of Sale)
- CI (Continuous Integration)

Целевая аудитория:

Владельцы розничных точек, которые хотят автоматизировать продажи и учет товара на складах. 

ПО включает в себя:

- складской учет
- продажа товара
- оформление и списане товара
- печать ценников
- считывание по штрихкоду
- график продаж
- печать документов
